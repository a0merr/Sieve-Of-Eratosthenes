## 📌 Project Overview
This project implements the **Sieve of Eratosthenes algorithm** to generate all prime numbers less than or equal to a given number `n`.  

The Sieve of Eratosthenes is an **efficient algorithm** for finding prime numbers by iteratively marking multiples of each prime starting from 2.  

In this script:  
- The upper limit `n` is currently set to **40,000,000**  
- The program generates and prints **all prime numbers up to `n`**
